## Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for API calls

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Sagarahlawat001/weather-1.git
cd weather-1
```

2. Open the application:
   - Simply open `index.html` in your web browser

## How to Use

1. Open the application in your browser
2. Allow location access or enter a city name
3. View the current weather information
4. Weather data updates in real-time

## API Integration

This application uses a weather API to fetch real-time weather data. Make sure you have:
- Valid API credentials configured in `app.js`
- An active internet connection
